# MARTE - აპლიკაციის აღწერა

## 🚗 რა არის MARTE?

**MARTE** არის სრული პლატფორმა მანქანის მფლობელებისთვის, რომელიც აერთიანებს ყველა ავტომობილთან დაკავშირებულ სერვისს ერთ ადგილზე. ეს არის "ყველაფერი მანქანისთვის ერთ ადგილზე" - სრული ეკოსისტემა საქართველოში.

---

## 📱 ძირითადი ფუნქციები

### 1. 🔍 **ნაწილების მარკეტპლეისი**
- მანქანის ნაწილების ძიება მარკის, მოდელისა და წლის მიხედვით
- კავშირი ავტონაწილების მაღაზიებთან და დისმანტლერებთან
- AI-ით გაძლიერებული რეკომენდაციები თქვენი მანქანისთვის შესაფერისი ნაწილებისთვის
- რეალურ დროში ფასების და ხელმისაწვდომობის ინფორმაცია
- პირდაპირი კომუნიკაცია გამყიდველებთან

### 2. 🧼 **ავტოსამრეცხაო სერვისები**
- ავტოსამრეცხაოების მოძიება და ბუკინგი
- სერვისის დეტალები, ფასები და ხელმისაწვდომობა
- რეიტინგებზე და მიმოხილვებზე დაფუძნებული რეკომენდაციები
- ბუკინგის სისტემა თარიღისა და დროის არჩევით
- სერვისის ისტორია და ქვითრები

### 3. 🛠️ **მექანიკოსების სერვისები**
- კვალიფიცირებული მექანიკოსების მოძიება თქვენს რაიონში
- ბუკინგი რემონტისა და მოვლისთვის
- მექანიკოსების პროფილები, სპეციალიზაციები და რეიტინგები
- რეალურ დროში კომუნიკაცია მექანიკოსებთან
- სერვისის მიმდინარეობის თვალყურის დევნება

### 4. 🚗 **მანქანების გაქირავება**
- მანქანების გაქირავების კატალოგი
- ფილტრაცია კატეგორიით, ფასით, ტრანსმისიით
- ბუკინგის სისტემა
- მანქანის დეტალური ინფორმაცია და ფოტოები

### 5. 🏪 **მაღაზიების დირექტორია**
- ავტომობილთან დაკავშირებული მაღაზიების სრული ბაზა
- კატეგორიები: ავტონაწილები, რემონტის სერვისები, დისმანტლერები
- მაღაზიების პროფილები, კონტაქტები, სამუშაო საათები
- ძიება მარკის, მოდელის, წლის მიხედვით
- რეიტინგები და მიმოხილვები

### 6. 🔧 **გარაჟის მენეჯმენტი**
- ციფრული გარაჟი რამდენიმე მანქანის მართვისთვის
- მანქანის ინფორმაცია: მარკა, მოდელი, წელი, ნომერი, ფერი
- მოვლის შეხსენებები და სერვისის ისტორია
- საწვავის მოხმარების თვალყურის დევნება
- მანქანისთვის სპეციფიკური რეკომენდაციები

### 7. 💬 **რეალურ დროში მესიჯინგი**
- უსაფრთხო ჩეთი მომხმარებლებსა და სერვისის მიმწოდებლებს შორის
- WebSocket-ზე დაფუძნებული რეალურ დროში კომუნიკაცია
- Push შეტყობინებები ახალი მესიჯებისთვის
- საუბრების ისტორია და ფაილების გაზიარება

### 8. 🤖 **AI-ით გაძლიერებული რეკომენდაციები**
- ინტელექტუალური მატჩინგის სისტემა ნაწილებისა და სერვისებისთვის
- ნდობის ქულა მანქანის თავსებადობის მიხედვით
- პერსონალიზებული რეკომენდაციები მომხმარებლის ისტორიის მიხედვით
- ჭკვიანი შეტყობინებები როცა შესაფერისი ნაწილები ხელმისაწვდომი ხდება

### 9. 💳 **გადახდის ინტეგრაცია**
- BOG (საქართველოს ბანკი) გადახდის გეითვეი
- უსაფრთხო გადახდების დამუშავება სერვისებისა და ნაწილებისთვის
- გადახდების ისტორია და ქვითრები
- მრავალი გადახდის მეთოდის მხარდაჭერა

### 10. 📊 **CarFAX ინტეგრაცია**
- მანქანის ისტორიის რეპორტები
- ავარიებისა და დაზიანებების ჩანაწერები
- მფლობელობის ისტორია
- სერვისის ჩანაწერები

### 11. 💰 **ფინანსირების ოფციები**
- ავტომობილის ფინანსირების განაცხადი
- სესხის კალკულატორი და ელიგიბილობის შემოწმება
- ლიდების მენეჯმენტის სისტემა

### 12. 👥 **კომუნიტეტის ფუნქციები**
- მომხმარებლის პოსტები და სტორები
- კომუნიტეტის ჩართულობა (ლაიკები, კომენტარები)
- გამოცდილებებისა და მიმოხილვების გაზიარება

### 13. ⛽ **საწვავის ფასები**
- საწვავის ფასების მონიტორინგი
- ბენზინგასამართი სადგურების რუკა
- ფასების შედარება

### 14. 🚛 **ევაკუატორის სერვისები**
- ევაკუატორის მოძიება და ბუკინგი
- სერვისის დეტალები და ფასები

## 🎯 ვინ იყენებს MARTE-ს?

1. **მანქანის მფლობელები** - ვინც ეძებს ნაწილებს, სერვისებს და მოვლას
2. **სერვისის მიმწოდებლები** - ავტოსამრეცხაოები, ავტონაწილების მაღაზიები, რემონტის სერვისები
3. **მექანიკოსები** - დამოუკიდებელი მექანიკოსები და სერვისის ტექნიკოსები
4. **დისმანტლერები** - ბიზნესები რომლებიც ყიდიან გამოყენებულ ნაწილებს

## 💡 რატომ MARTE?

### ⏱️ **დროის დაზოგვა**
- სწრაფი წვდომა მრავალ სერვისის მიმწოდებელზე ერთ პლატფორმაზე

### 💰 **ფასების შედარება**
- შეადარეთ ფასები და იპოვეთ საუკეთესო შეთავაზებები

### ⭐ **ხარისხის გარანტია**
- რეიტინგები და მიმოხილვები ეხმარება ინფორმირებული გადაწყვეტილების მიღებაში

### 🎯 **მოხერხებულობა**
- ბუკინგი, ნაწილების შეკვეთა და კომუნიკაცია მიმწოდებლებთან - ყველაფერი ერთ აპში

### 🔒 **ნდობა**
- დადასტურებული სერვისის მიმწოდებლები და უსაფრთხო გადახდების დამუშავება

### 🧠 **ინტელექტი**
- AI-ით გაძლიერებული რეკომენდაციები უზრუნველყოფს, რომ იპოვოთ ზუსტად ის, რაც გჭირდებათ

## 🛠️ ტექნიკური დეტალები

- **Frontend**: React Native (Expo) - iOS და Android
- **Backend**: NestJS (TypeScript)
- **Database**: MongoDB
- **Real-time**: WebSocket (Socket.io)
- **Notifications**: Firebase Cloud Messaging
- **Payments**: BOG Payment Gateway
- **Maps**: Google Maps API
