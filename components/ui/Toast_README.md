# Toast рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў

рЃџрЃљрЃЏрЃљрЃќрЃў рЃљрЃюрЃўрЃЏрЃўрЃарЃћрЃЉрЃБрЃџрЃў Toast рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў CarappX-рЃўрЃАрЃЌрЃЋрЃўрЃА.

## рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

- ­Ъје **4 рЃбрЃўрЃърЃў**: success, error, warning, info
- Рюе **рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў**: рЃерЃћрЃЏрЃЮрЃАрЃЋрЃџрЃљ, рЃњрЃљрЃЦрЃарЃЮрЃЉрЃљ, scale рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃў
- ­Ъј» **рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**: Context API-рЃўрЃЌ
- ­ЪЊ▒ **рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃЋрЃў**: рЃДрЃЋрЃћрЃџрЃљ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃќрЃЮрЃЏрЃљрЃќрЃћ
- ­Ъје **рЃџрЃљрЃЏрЃљрЃќрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў**: рЃдрЃўрЃљ UI рЃЌрЃћрЃЏрЃљ
- ­ЪЄг­ЪЄф **рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃћрЃюрЃљ**: рЃДрЃЋрЃћрЃџрЃљ рЃбрЃћрЃЦрЃАрЃбрЃў рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

### 1. Context-рЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў

```tsx
import { useToast } from '../../contexts/ToastContext';
```

### 2. Toast-рЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

```tsx
const { success, error, warning, info } = useToast();

// рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃљ
success('рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃљ!', 'рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ');

// рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
error('рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ!', 'рЃарЃљрЃдрЃљрЃф рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЏрЃЮрЃ«рЃЊрЃљ');

// рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ
warning('рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ!', 'рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃўрЃЌ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃЌ');

// рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
info('рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ', 'рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ');
```

### 3. рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

```tsx
const { showToast, hideToast, hideAllToasts } = useToast();

// рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў Toast-рЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ
showToast({
  type: 'success',
  title: 'рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃљ!',
  message: 'рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ',
  duration: 5000,
  position: 'top'
});

// Toast-рЃўрЃА рЃЊрЃљрЃЏрЃљрЃџрЃЋрЃљ
hideToast('toast-id');

// рЃДрЃЋрЃћрЃџрЃљ Toast-рЃўрЃА рЃЊрЃљрЃЏрЃљрЃџрЃЋрЃљ
hideAllToasts();
```

## рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў

### ToastType
- `success` - рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ
- `error` - рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ
- `warning` - рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃљ
- `info` - рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ

### ToastProps
- `id: string` - рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў ID
- `type: ToastType` - Toast-рЃўрЃА рЃбрЃўрЃърЃў
- `title: string` - рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃбрЃћрЃЦрЃАрЃбрЃў
- `message?: string` - рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃбрЃћрЃЦрЃАрЃбрЃў
- `duration?: number` - рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ (рЃЏрЃўрЃџрЃўрЃгрЃљрЃЏрЃћрЃЉрЃерЃў)
- `position?: 'top' | 'bottom'` - рЃърЃЮрЃќрЃўрЃфрЃўрЃљ рЃћрЃЎрЃарЃљрЃюрЃќрЃћ

## рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў

- **рЃерЃћрЃЏрЃЮрЃАрЃЋрЃџрЃљ**: translateY, opacity, scale
- **рЃњрЃљрЃЦрЃарЃЮрЃЉрЃљ**: translateY, opacity, scale
- **Spring рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ**: scale рЃћрЃцрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА
- **Timing рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљ**: translateY рЃЊрЃљ opacity-рЃўрЃАрЃЌрЃЋрЃўрЃА

## рЃАрЃбрЃўрЃџрЃћрЃЉрЃў

- **Success**: рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃцрЃћрЃарЃћрЃЉрЃў
- **Error**: рЃгрЃўрЃЌрЃћрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃў  
- **Warning**: рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃў
- **Info**: рЃџрЃБрЃарЃ»рЃў рЃцрЃћрЃарЃћрЃЉрЃў

## рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў

```tsx
import React from 'react';
import { View } from 'react-native';
import Button from './Button';
import { useToast } from '../../contexts/ToastContext';

const MyComponent = () => {
  const { success, error } = useToast();

  const handleSuccess = () => {
    success('рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃљ!', 'рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ');
  };

  const handleError = () => {
    error('рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ!', 'рЃарЃљрЃдрЃљрЃф рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ рЃЏрЃЮрЃ«рЃЊрЃљ');
  };

  return (
    <View>
      <Button title="рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃљ" onPress={handleSuccess} />
      <Button title="рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ" onPress={handleError} />
    </View>
  );
};
```
